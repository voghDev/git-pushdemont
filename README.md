# git-pushdemont
Custom Git command that marks your commits as pushed, then reverts the process after 8 seconds

# Installing

    sudo cp git-pushdemont /bin
    chmod a+x git-pushdemont
    
Then go to your favorite project and you can execute

    git pushdemont
